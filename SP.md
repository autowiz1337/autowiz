## 3. Aesthetic & UI Directives

To match the high-tech nature of the JSON output:

1.  **"Glass & Neon" aesthetic:** Dark mode base (`#020617`) with glowing accents to represent AI processing power.
2.  **Data Visualization:** Use code snippets or JSON blocks as decorative elements (e.g., `{ "trigger": "Status" }`) fading into the background to emphasize the tech.
3.  **Micro-Interactions:** When hovering over features, simulate the "Scanning" effect seen in the `PaidCheckout` flow.
